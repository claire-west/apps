The technical stuff

This is a container for modular web applications. A core JavaScript library loads additional HTML, CSS, and JavaScript modules dynamically on an as-required basis.

One module handles user login via Google's OAuth 2.0 API, another handles URL changes to navigate from page to page.

Really it does a lot of the same things a library like Angular would offer, but I did it myself, implementing specific features based on my needs.

Use the menu at the top left to navigate to an app, or use the tiles below to learn more about each application. Some applications require sign-in with your Google account using the button in the top right.
